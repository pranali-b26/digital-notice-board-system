Digital Notice Board System

Project Description

The Digital Notice Board System is a Python-based application that allows administrators to manage notices digitally.
Admins can add, update, delete, and search notices, while users can view notices sorted by date or category.

This system replaces the traditional notice board with a simple digital solution.

Features

- Admin Login Authentication
- Add New Notice
- View All Notices in Table Format
- Display Latest Notice
- Search Notice by Title or Category
- Update Existing Notice
- Delete Notice
- Store Notices using JSON file

Technologies Used

- Python
- JSON (for data storage)
- VS Code (for development)

Project Structure

digital-notice-board-system
│
├── notice_board.py
└── notices.json

How to Run the Project

1. Install Python on your system.
2. Open the project folder in VS Code.
3. Run the following command in the terminal:

python notice_board.py

4. Login using:

Username: admin
Password: 1234

Advantages

- Simple and easy to use
- Notices stored permanently in JSON file
- Quick search and update functionality
- Reduces paper-based notice boards

Conclusion

The Digital Notice Board System provides a simple and efficient way to manage notices digitally. It improves communication by allowing administrators to quickly post and manage notices while users can easily view important information.
